# GroAsk

macOS 菜单栏 AI 启动器，`⌥Space` 直达 ChatGPT / Claude / Gemini。

## 下载

[**下载最新版本**](https://github.com/ThinkerJack/groask-release/releases/latest/download/GroAsk.dmg)

前往 [Releases](https://github.com/ThinkerJack/groask-release/releases) 查看所有版本。

## 安装

1. 下载 `GroAsk.dmg`
2. 打开 DMG，将 GroAsk 拖入 Applications 文件夹
3. 首次打开时如遇安全提示，前往「系统设置 → 隐私与安全性」允许运行

## 系统要求

- macOS 13.0 (Ventura) 及以上
